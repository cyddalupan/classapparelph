Step 3: Common Fixes

Try These Solutions
1. Date Format Issue: - HTML date input: YYYY-MM-DD (should work) - Try: 2026-02-28 (not 28/02/2026) 2. Amount Format Issue: - Should be: 1000.50 (not 1,000.50) - Decimal separator: . (dot) not , (comma) 3. CSRF Token Missing: - Check console for "CSRF token mismatch" - Ensure _token field is included 4. Character Encoding: - Avoid special characters in description - Try plain text first 5. Test with Minimal Data: { "date": "2026-02-28", "amount": 100.50, "category": "office_supplies", "status": "pending", "description": "Test expense", "_token": "your-csrf-token-here" }
