# User Guide - Sales Staff

## 📋 Introduction
Welcome to the CLASS Apparel PH Sales System! This guide will help you use the application to manage sales, track orders, and serve customers efficiently.

**Application URL:** https://app.classapparelph.com

## 👤 Getting Started

### 1. Login
1. Go to https://app.classapparelph.com/login
2. Enter your email and password
3. Click "Login"

### 2. Dashboard Overview
After login, you'll see the main dashboard with:
- **Today's Sales** - Total sales for today
- **Pending Orders** - Orders needing attention
- **Low Stock Alerts** - Products running low
- **Recent Activity** - Latest sales and updates

## 🛒 Creating a Sale

### Step 1: Start New Sale
1. Click "New Sale" button in the top navigation
2. Or go to `/sales/create-quick`

### Step 2: Select Order Type
You'll see four main order types:

#### A. iPrint (Standard Printing)
- **DTF Printing** - Direct-to-film transfers
- **Lanyard Printing** - Lanyard customization
- **Tarpaulin Printing** - Large format printing
- **Sublimation Printing** - Fabric printing
- **Embroidery** - Thread embroidery
- **Other Items** - Miscellaneous printing

#### B. Consol (Consolidated Services)
- Bulk printing services
- Simplified pricing
- Faster checkout

#### C. Class (Training/Workshops)
- DTF Training
- Sublimation Training
- Business Workshops
- Custom Classes

#### D. Cinco (Package Deals)
- Service bundles
- Package discounts
- Special promotions

### Step 3: Fill Order Details

#### For DTF Orders:
1. **Select Print Type:**
   - Standard Print
   - Shirt Print
   - Hoodie Print
   - Other

2. **Add Sizes and Quantities:**
   - Click "Add Size" button
   - Select size (8x10, 12x18, etc.)
   - Enter quantity
   - Repeat for multiple sizes

3. **Additional Options:**
   - Select colors
   - Add special instructions
   - Upload design files (if needed)

#### For Other Printing Types:
Similar process with type-specific options.

### Step 4: Add to Order
1. Click "Add to Order" button
2. Item appears in the Order Items table
3. You can add multiple items to the same order

### Step 5: Customer Information
1. **Existing Customer:**
   - Start typing customer name
   - Select from dropdown
   - Customer details auto-filled

2. **New Customer:**
   - Click "New Customer"
   - Fill in name, email, phone
   - Customer saved for future orders

### Step 6: Payment
1. **Payment Method:**
   - Cash
   - Credit Card
   - Bank Transfer
   - GCash
   - PayMaya

2. **Payment Status:**
   - Paid (full payment)
   - Partial (deposit paid)
   - Pending (to pay later)

3. **Discounts:**
   - Enter discount amount or percentage
   - System calculates final total

### Step 7: Complete Sale
1. Review order summary
2. Click "Complete Sale"
3. Print receipt (optional)
4. Email receipt to customer (optional)

## 📊 Managing Orders

### View All Orders
1. Go to "Sales" → "All Orders"
2. Filter by date, status, customer
3. Search by order number or customer name

### Edit Order
1. Find order in list
2. Click "Edit" button
3. Make changes
4. Click "Update Order"

**Note:** Only pending orders can be edited. Paid orders require manager approval.

### Cancel Order
1. Find order in list
2. Click "Cancel" button
3. Select reason for cancellation
4. Confirm cancellation

### Print Receipt/Invoice
1. Open order details
2. Click "Print Receipt"
3. Select receipt format
4. Print or save as PDF

## 📦 Product Management

### View Products
1. Go to "Products" → "All Products"
2. See product list with prices and stock
3. Filter by category or search by name

### Check Stock Levels
- **Green:** Good stock (> minimum)
- **Yellow:** Low stock (≤ minimum)
- **Red:** Out of stock (0)

### Product Details
Click any product to see:
- Product description
- Current stock
- Price history
- Sales history
- Product images

## 💰 Expense Tracking

### Add Expense
1. Go to "Expenses" → "New Expense"
2. Select category:
   - Materials
   - Utilities
   - Rent
   - Salaries
   - Marketing
   - Other

3. Enter expense details:
   - Date
   - Description
   - Amount
   - Payment method
   - Upload receipt (optional)

4. Click "Save Expense"

### View Expenses
1. Go to "Expenses" → "All Expenses"
2. Filter by date, category, amount
3. View monthly expense reports

## 🖨️ Printing Services Guide

### DTF Printing Process
1. **Customer Consultation:**
   - Discuss design requirements
   - Show sample products
   - Provide price quote

2. **Order Entry:**
   - Select DTF from iPrint menu
   - Choose print type
   - Specify sizes and quantities
   - Note special requirements

3. **Production:**
   - Design approval
   - Film printing
   - Heat pressing
   - Quality check

4. **Delivery/Pickup:**
   - Schedule pickup/delivery
   - Update order status
   - Collect payment if pending

### Sublimation Printing
**Best for:** Mugs, tumblers, polyester fabrics
**Process:** Similar to DTF but with different materials

### Embroidery
**Best for:** Company logos, uniforms, caps
**Process:** Digitize design, set up machine, embroider

### Tarpaulin Printing
**Best for:** Banners, signage, event materials
**Process:** Large format printing, grommet installation

## 🆘 Troubleshooting

### Common Issues

#### 1. Can't Login
- Check email and password
- Caps Lock might be on
- Contact administrator if locked out

#### 2. Form Not Submitting
- Check all required fields (marked with *)
- Ensure internet connection is stable
- Try refreshing the page

#### 3. Slow Performance
- Clear browser cache
- Close other tabs/applications
- Check internet speed

#### 4. Printing Issues
- Check printer connection
- Ensure correct paper size
- Try printing as PDF first

### Error Messages

#### "Invalid Customer"
- Customer might be deleted
- Try creating new customer
- Contact administrator

#### "Insufficient Stock"
- Product stock is too low
- Check alternative products
- Update stock or create backorder

#### "Payment Failed"
- Check payment details
- Try different payment method
- Contact payment provider
